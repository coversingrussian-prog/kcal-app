# КалорияСкан — как запустить

Приложение готово. Осталось связать 3 вещи: бот, ключ Gemini, деплой.
Всё бесплатно. Займёт ~15 минут.

---

## Шаг 1. Получи бесплатный ключ Gemini

1. Зайди на **https://aistudio.google.com/apikey**
2. Войди через Google-аккаунт.
3. Нажми **Create API key** → скопируй ключ (выглядит как `AIza...`).
4. Сохрани его — понадобится на шаге 3.

Бесплатный лимит Gemini: ~1500 запросов в день. На старте хватит с запасом.

---

## Шаг 2. Задеплой на Netlify

Вариант А — через сайт (проще):
1. Зайди на **https://app.netlify.com** → зарегистрируйся (можно через GitHub/Google).
2. Netlify Drop работает только для статики. Для функций нужен способ ниже:
3. Нажми **Add new site → Deploy manually**, перетащи ВСЮ папку `kcal-app`.
   - Если функция не подхватится — используй Вариант Б.

Вариант Б — через GitHub (надёжнее для функций):
1. Создай репозиторий на github.com, залей туда содержимое папки `kcal-app`.
2. В Netlify: **Add new site → Import from Git** → выбери репозиторий.
3. Настройки сборки Netlify возьмёт из `netlify.toml` автоматически. Жми Deploy.

После деплоя получишь адрес вида `https://твой-сайт.netlify.app`.

---

## Шаг 3. Впиши ключ Gemini в Netlify

1. В панели сайта: **Site configuration → Environment variables**.
2. Нажми **Add a variable**:
   - Key: `GEMINI_API_KEY`
   - Value: твой ключ `AIza...` со шага 1
3. Сохрани и нажми **Redeploy** (Deploys → Trigger deploy).

Ключ теперь на сервере, в коде фронтенда его нет — украсть нельзя.

---

## Шаг 4. Создай Telegram-бота и Mini App

1. В Telegram открой **@BotFather**.
2. Отправь `/newbot` → задай имя и username бота.
3. BotFather выдаст токен — сохрани (пригодится, если делать бота-обёртку).
4. Отправь `/newapp` → выбери своего бота.
5. BotFather попросит:
   - Название, описание, картинку (640×360).
   - **Web App URL** → вставь адрес Netlify (`https://твой-сайт.netlify.app`).
6. Готово — BotFather даст ссылку на Mini App (`t.me/твой_бот/имя_аппа`).

---

## Шаг 5. Проверь

Открой ссылку Mini App в Telegram → сфоткай еду → получи калории.

## Реклама (когда будешь готов)

В коде `index.html` есть блок с id `adSlot` — это готовое место под рекламу.
Варианты монетизации:
- **Telegram Ads / Adsgram** — рекламная сеть специально для Telegram Mini Apps. Регистрируешься, вставляешь их скрипт в слот.
- **Прямые рекламодатели** — сам продаёшь место (текст + ссылку) в блоке `adTitle`/`adDesc`.

Начни с прямых — ты сам сказал, что с рекламодателями разберёшься.
